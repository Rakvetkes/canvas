# Lelyetia 生物群系草方块颜色配置

## 颜色方案总览

已为所有11个生物群系添加自定义草方块和树叶颜色。配色遵循以下原则：
- **倒置群系**：比对应正常群系更亮
- **河流**：与金色树林保持一致
- **飞升峰顶**：比要塞山脉略亮

## JSON 配置示例

每个生物群系的 `effects` 部分现在都包含：

```json
"effects": {
  "sky_color": 10212295,
  "fog_color": 13095075,
  "water_color": 5221530,
  "water_fog_color": 3109734,
  "grass_color": 13937968,      // 草方块和草的颜色
  "foliage_color": 13937968,    // 树叶的颜色
  "mood_sound": {
    "sound": "minecraft:ambient.cave",
    "tick_delay": 6000,
    "block_search_extent": 8,
    "offset": 2.0
  }
}
```

## 颜色亮度对比

### 正常 vs 倒置群系
- **要塞山脉** (0x8B9B8B) → **倒置圣殿** (0xA0B0A0) ⬆️ +21%
- **金色树林** (0xD4A930) → **倒置金色树林** (0xE8BD40) ⬆️ +15%
